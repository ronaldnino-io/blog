Infraestructura de Software

En la última década se han desarrollado tecnologías que nos permite tener un ambiente totalmente virtualizado donde podemos mantener agrupados de manera lógica recursos de computación, red y almacenamiento permitiéndonos aprovisionamiento de recursos basado en políticas y automatización de TI.

Desarrollo de Software

Es una disciplina que permite el desarrollar productos de software. Existen varios modelos que tienen como objetivo establecer procesos para el desarrollo del software y cada uno describe un enfoque diferente para las actividades que tiene lugar durante el proceso.

UX & UI

Entender cómo las personas perciben un producto o servicio al interactuar con ellos, es fundamental para lograr diseños útiles usables y deseables.
